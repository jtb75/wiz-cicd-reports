# Example Output Files

This directory contains example outputs generated by the Wiz CI/CD reporting tool.

## Files

### Interactive Dashboard
- **dashboard_example.html** - Multi-tab interactive dashboard with charts and filtering
  - Open in a browser to see the full interactive experience
  - Features: Executive Summary, Detailed Reporting, Top 10 Riskiest Apps
  - Includes filters, drill-down navigation, and PDF export

### CSV Reports
- **executive_summary_example.csv** - High-level scan statistics
  - Scan verdicts (Passed/Failed/Warned)
  - Finding counts by type and severity

- **daily_trends_example.csv** - Daily trend data
  - Daily scan counts and verdicts
  - Daily finding counts by severity
  - Useful for tracking improvement over time

## Example Data

These examples are based on real Wiz CI/CD scan data showing:
- **1,228 total scans** across 30 days
- **541 passed (44.1%)**, 297 failed (24.2%), 384 warned (31.3%)
- **98,204 vulnerabilities** detected
- **1,362 secrets** found
- **16,883 IaC issues** identified
- **2,664 sensitive data findings**

## How to Generate Your Own

```bash
# Generate CSV reports
python generate_reports.py

# Generate interactive dashboard
python generate_dashboard_multitab.py

# Your outputs will be in the output/ directory
```
